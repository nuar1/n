# Sliding Door Experts - Mobile-Optimized Repair Website

A professional, mobile-first sliding door repair website built with modern web technologies and optimized for local SEO and conversions.

## 🚪 Features

### Core Functionality
- **Mobile-First Design** - Fully responsive with mobile CTAs
- **24/7 Emergency Services** - Prominent emergency contact options
- **Service Area Coverage** - North Fort Myers, Cape Coral, Naples, and surrounding areas
- **Interactive Quote Form** - Real-time validation and ZIP code checking
- **Customer Testimonials** - Auto-rotating carousel with real customer reviews
- **Live Chat Integration** - Floating chat bubble for instant support

### Technical Features
- **Bootstrap 5.3** - Modern responsive framework
- **Google Fonts** - Montserrat & Open Sans typography
- **FontAwesome 6** - Professional iconography
- **Swiper.js** - Smooth testimonial carousel
- **Progressive Enhancement** - Works without JavaScript
- **SEO Optimized** - Meta tags, structured content, semantic HTML
- **Performance Optimized** - Lazy loading, WebP images, minified assets

### Interactive Elements
- Sticky header with scroll behavior
- Hamburger menu with full-screen overlay
- Pulsing phone button animations
- Service card hover effects
- Form validation with real-time feedback
- ZIP code service area validation
- Mobile-specific touch interactions

### Google Analytics Setup
Add your GA4 tracking code before the closing `</head>` tag:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```
